🚀 COMO USAR:

1️⃣  DESCOMPACTAR:
    • Windows: Clique direito > Extrair
    • Mac: Duplo clique no ZIP
    • Linux: unzip landing-page-terapia.zip

2️⃣  NAVEGAR PARA A PASTA:
    cd landing-page-terapia

3️⃣  COPIAR ARQUIVO DE AMBIENTE:
    cp .env.local.example .env.local

4️⃣  EDITAR .env.local:
    • Abrir em seu editor (VSCode, Sublime, etc)
    • Preencher seus dados:
      - PROFESSIONAL_NAME (seu nome)
      - PROFESSIONAL_PHONE (seu WhatsApp)
      - PROFESSIONAL_EMAIL (seu email)

5️⃣  INSTALAR E RODAR:
    npm install
    npm run dev

6️⃣  ACESSAR:
    Abrir navegador: http://localhost:3000
